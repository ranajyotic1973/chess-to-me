# Free Chess Databases For Integration

## 1) Lichess Database Dumps

- Website: https://database.lichess.org/
- Content: very large monthly game dumps.
- License notes: site states game exports are free and includes licensing notes on page.
- Format: compressed archives (for large-scale datasets).

## 2) PGN Mentor Collections

- Website: https://www.pgnmentor.com/files.html
- Content: player/event opening collections as `.pgn`.
- Format: direct `.pgn` files suitable for immediate loading in this app.

## 3) The Week in Chess (TWIC)

- Website: https://theweekinchess.com/twic
- Content: weekly tournament game collections.
- Format: downloadable archives containing PGN.

## Practical Recommendation For This App

- Use PGN Mentor/TWIC for easy direct import in the Reference Games tab.
- Use Lichess dumps for very large datasets and pre-processing pipelines.
